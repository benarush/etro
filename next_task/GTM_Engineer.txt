
# AI-Powered Lead Outreach & Scoring

## 📄 Overview
Your task is to simulate a mini AI-powered marketing automation workflow for Stampli. You will work with a sample dataset of leads and build a script that:
1. Generates personalized outreach emails for Stampli’s received leads using the OpenAI API.
2. Scores leads based on job relevance and engagement.
3. Sends results to a simulated CRM endpoint.

---

## 📦 What We Provide
- A sample CSV file (`leads.csv`) with the following fields:
  - `name`
  - `company`
  - `job_title`
  - `engagement_activity` (e.g., "Visited pricing page", "Attended webinar")
  - `industry`



---

## ✅ What You Should Build
Please deliver a Python script or Jupyter Notebook that does the following:

1. Parse the CSV** and read each lead.
2. Leverages an LLM to craft concise, personalized outreach emails for Stampli’s inbound leads.
   - Base the message on job title and engagement activity.
3. Assign a lead score (1–100)** using some scoring logic.
4. Log each lead** (name, email content, score).

---

## 🧪 What We're Looking For
- Clarity of logic and code quality
- Smart use of prompts and personalization
- Sensible lead scoring logic
- Simple, readable documentation
- Quick and accurate

---

## 📄 What to show
- Your code
- A brief README or Google Doc explaining:
  - Prompt structure (show at least one example)
  - Scoring rules
  - Any assumptions or limitations
